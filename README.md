Company System - Java Project

A Java-based Company Management System designed to manage employees, departments, salaries, and company operations. This project demonstrates Object-Oriented Programming (OOP) principles and basic management system functionality.

Features
Employee management
Department management
Salary tracking
Employee role assignment
Search and display employee records
Console-based user interface
Technologies Used
Java
Object-Oriented Programming (OOP)
Java Collections Framework
File Handling (optional)
OOP Concepts Implemented
Classes and Objects
Inheritance
Encapsulation
Polymorphism
Method Overriding
Constructors
Access Modifiers
